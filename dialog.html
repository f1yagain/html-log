📌 Project Profile

项目名称
“光影随行” — 环境自适应夜灯
一句话产品定义
在暗光环境下检测到声音（如脚步声、拍手）自动亮起5秒，为夜间活动提供临时照明，避免强光刺眼。

老年人起夜、宿舍楼道、婴儿房 — 避免摸黑开灯，也无需声控开关常亮打扰他人。

💻 核心代码 (ESP32-S2, 峰值检测 + 双阈值触发)

#include <FastLED.h>

#define LIGHT_PIN     4
#define SOUND_PIN     5
#define LED_PIN       18
#define NUM_LEDS      8
#define BRIGHTNESS    128
#define LED_COLOR     CRGB(255,180,100)   // 暖白色护眼

#define LIGHT_DARK_THRESHOLD   1800
#define SOUND_PEAK_THRESHOLD   35
#define HOLD_TIME_MS           5000

CRGB leds[NUM_LEDS];
bool isLightOn = false;
unsigned long lastTrigger = 0;

int getSoundIntensity(int pin, int sampleWindowMs = 40) {
    unsigned long start = millis();
    int signalMin = 4095, signalMax = 0;
    while (millis() - start < sampleWindowMs) {
        int sample = analogRead(pin);
        if (sample < signalMin) signalMin = sample;
        if (sample > signalMax) signalMax = sample;
    }
    int peak = signalMax - signalMin;
    if (peak < 10) peak = 10;
    int loudness = map(peak, 10, 2000, 0, 100);
    return constrain(loudness, 0, 100);
}

void setup() {
    Serial.begin(115200);
    pinMode(LIGHT_PIN, INPUT);
    pinMode(SOUND_PIN, INPUT);
    FastLED.addLeds<WS2812, LED_PIN, GRB>(leds, NUM_LEDS);
    FastLED.setBrightness(BRIGHTNESS);
    turnOff();
}

void loop() {
    int lightVal = analogRead(LIGHT_PIN);
    int loud = getSoundIntensity(SOUND_PIN, 40);
    bool isDark = (lightVal > LIGHT_DARK_THRESHOLD);
    bool isLoud = (loud > SOUND_PEAK_THRESHOLD);

    if (isDark && isLoud) {
        lastTrigger = millis();
        if (!isLightOn) turnOn();
    }
    if (isLightOn && (millis() - lastTrigger >= HOLD_TIME_MS)) turnOff();
    delay(20);
}

🤔 Reflections · 硬核Bug与解决之道

🐞 Bug #1 声音模块死区
现象：KY-037的AO读数几乎不变，保持在750±5，拍手无反应。
原因：① 蓝色电位器处于中间位置，增益极小；② 原代码使用delay(50) + analogRead，错过声音瞬时峰值。
解决：李佳诚调节电位器直到LED指示灯在安静时刚熄灭；赵翰林重写getSoundIntensity()，在采样窗口内记录信号峰峰值，映射到0~100响度。
✨ 经验：对于动态传感器，峰值检测比瞬时采样可靠10倍。
🐞 Bug #2 灯带点亮瞬间读数跳变(4000)导致误触发
现象：灯带点亮时声音读数偶尔冲到4095，触发逻辑乱。
原因：灯带大电流导致地线弹跳，ESP32 ADC参考电位瞬时波动。
解决：李佳诚重新布线——传感器GND单独接ESP32 GND，灯带GND直接回外部电源负极，并在ESP32 GND与外部电源GND之间用粗线短接。同时在灯带DIN串联220Ω电阻抑制反射。杨欣翰协助验证稳定性。
✨ 教训：混合电路务必“星形一点共地”，大电流路径与小信号路径分离。
🐞 Bug #3 光强阈值逻辑反直觉
现象：本来暗环境应该点亮，反而亮环境触发了。
原因：MH光强传感器输出与光照成反比（越亮数值越小）。
解决：判断条件改为 lightValue > LIGHT_DARK_THRESHOLD 表示暗。并在注释中明确标记（赵翰林修正逻辑）。
✨ 收获：阅读模块手册或实测输出特性比凭直觉更重要。
🧩 结构件配合问题
现象：早期外壳传感器开孔偏差导致光强读数遮挡。
解决：万里驹根据实测传感器位置迭代第二版模型，增加导光槽和声音采集孔，杨欣翰协助进行光路测试。
✨ 团队感悟：软硬件与结构并行迭代，快速原型 + 实测修正效率最高。

总体反思： 李佳诚（硬件/电路）、赵翰林（软件）、万里驹（结构）、杨欣翰（测试/文档）四人分工明确，从最初不稳定的采样到稳定的峰值检测+共地优化+精密外壳，最终产品在凌晨测试中可靠性超过95%。HTML日志完整记录了每一次失败与迭代，成为答辩的有力证据。
